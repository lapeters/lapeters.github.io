This project was built to help me cut my teeth on Vue.js and JavaScript frameworks in general. It uses Axios to makes simple calls to the Accuweather third-party API in order to display the current weather conditions of a searched location. The search input accepts either city name or zipcode. Then the fighting begins!

Game is a turn-based battle simulator. Global data store used for search results, player health, and attack values for better control and reliability. Layout for app uses a custom-built Flexbox grid system created with SASS variables, mixins, and functions.
